# SAASY-MARKDOWN

SAASY Markdown npm lib

### Step 4: Install package inside your project

```sh
npm install https://github.com/SaaSy-Labs/sassy-markdown.git
```

## Publishing a new version for lib maintener

```sh
npm run rollup
```
